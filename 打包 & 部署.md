# 打包

## 打包
直接使用python运行一键生成分发资源.py即可
### 打包版安装
直接运行AppInstallRun.bat
将会自动安装uv（uv是本地的zip包，不存在网络问题）和cpython与所有依赖环境，并自动启动一次faa
有时下载存在一定网络问题（内置镜像发癫不稳定）导致安装失败，重新运行或更换镜像即可
### 正式版
安装后直接运行FAA.exe
### 调试版
安装后直接运行debug_run.bat


## 环境迁移

### uv安装

    #自行安装uv，网上教程很多此处不细述，或者把plugins文件夹下的文件移动到根目录直接运行AppInstallRun.bat

### 添加第三方库

    uv add requests

### 根据配置文件一键安装python库环境到本地

    uv sync

## git抽风无法提交到github

### 在git仓库打开终端 并输入下文

    git config --global --unset http.proxy
    git config --global --unset https.proxy

    git config --global http.proxy http://127.0.0.1:10809
    git config --global https.proxy http://127.0.0.1:10809
